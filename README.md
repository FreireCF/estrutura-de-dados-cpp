## 🚀 Como compilar

Você pode compilar qualquer arquivo `.cpp` com o `g++`:

```bash
g++ pilhas/pilha_vetor.cpp -o pilha
./pilha
```

Se preferir, use o Makefile (em breve no projeto):

```bash
make pilha
```
